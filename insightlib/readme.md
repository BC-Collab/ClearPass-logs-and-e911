These are library files and utility programs being used by
the Aruba ClearPass Policy Manager Insight Log Extract programs.

/var/app/insightlibs    - This directory contains common programs and
                          libraries used for these other programs.
/var/app/insightexports - Exports log events into CEF text file format.
/var/app/insighttodb    - Exports log event informations into a database table.
/var/app/e911		- Uses the Insight Events Dataase table to build
                          reports of the movement of VoIP phone handsets.
/var/app/logcompress    - Compresses log files and deletes older versions.
